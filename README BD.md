#Recuperar la base de datos.

	1. Abrir SQL Management Studio 2014
	2. Estando dentro de SQL MS2014 hacer click derecho en la carpeta **base de datos**
	   y despues click en restaurar base de datos
	3. En la pestaña **General** hacer click en **Dispositivo** y despues en los **tres puntos** al lado
	4. Se abrira una pestaña con el siguiente texto
		**Seleccionar dispositivos de copia de seguridad**
		Dar click en agregar.
		Irnos a nuestra Ruta **D:** 
		Dar Click en **ASP.NET ENTITY FRAMEWORK**
		y elegir el archivo que se encuentra en esta carpeta con el nombre **BDPasaje.bak**


	**Nota:** Si el archivo no aparece, favor de copiarlo a la carpeta de **ASP.NET ENTITY FRAMEWORK**
